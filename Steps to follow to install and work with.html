Steps to follow to install and work with Django -----DJANGO INSTALLATION----- 1. Create a new folder on your desktop to install virtual environments 2. Open command prompt and type *virtualenv env* or *python3 -m venv My_env* to create virtual environment 3. Activate the virtual environment with the following command *...\scripts\activate* 4. install Django with the following command *pip install Django* 5. Start a project with the following command *django-admin startproject Project_Name* 6. Channge dir to the new project and run django server with the following command *python manage.py runserver* 7. Create a new app with the following command *python manage.py startapp *app_name* 8. Link your app with your project folder by opening settings of your project. type the following command in installed apps of your project *app_name.apps.app_nameConfig* -----URL ROUTING----- -----FEED COMPONENT-------------
